# auto-branch-pr

![](bot.gif)

> A GitHub App built with [Probot](https://github.com/probot/probot) that Auto creates Pull Request to a base branch when a destination changes

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
